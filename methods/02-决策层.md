# 决策层（#11-#20）——怎么选

---

## #11 二阶思维
- **来源**：西方（霍华德·马克斯）
- **核心问题**：后果的后果是什么
- **核心动作**：推演"然后呢？再然后呢？"

**思考流程**：
1. 一阶结果：这个决定的直接后果
2. 二阶结果：后果引发的后果
3. 三阶结果：最终会怎样
4. 评估每个结果的概率和影响

**关键提问**：
- 这个决定的直接后果是什么？
- 后果的后果是什么？
- 有没有意料之外的副作用？
- 长期和短期有什么冲突？

**示例**：
```
决策：为了省钱砍掉测试团队
一阶：成本降低 ✓
二阶：bug增加 → 客服成本上升 ✗；用户流失 → 收入下降 ✗
三阶：口碑恶化 → 获客成本上升 ✗；优秀员工离职 → 招聘成本上升 ✗
洞察：短期省钱，长期亏钱
```

**组合接口**：#12 阴阳平衡、#14 期望值思维、#51 反脆弱
**反面风险**：推演过远，失去行动力

## #12 阴阳平衡
- **来源**：华夏（《道德经》《易经》）
- **核心问题**：另一面是什么
- **核心动作**：强制看对立面，找太极融合点

**思考流程**：
1. 明确当前倾向/判断
2. 强制站在对立面思考
3. 找到对立面中的合理成分
4. 寻找"既A又B"的第三方案

**关键提问**：
- 如果我错了，正确的一面是什么？
- 对立面完全没道理吗？
- 有没有一种方式能同时满足两个需求？

**组合接口**：#11 二阶思维、#17 中庸之道、#19 反向思维
**反面风险**：变成和稀泥，不做选择

## #13 机会成本
- **来源**：西方（经济学基本原理）
- **核心问题**：放弃了什么
- **核心动作**：算清每个选择背后放弃的东西

**思考流程**：
1. 列出所有选项
2. 对每个选项，列出"选了它就放弃了什么"
3. 评估放弃的东西的价值
4. 选择机会成本最低的

**关键提问**：
- 选了A，我同时放弃了B的什么？
- 放弃的东西，有没有不可逆的？
- 5年后，放弃的东西还重要吗？

**组合接口**：#11 二阶思维、#14 期望值思维、#20 多路径对比
**反面风险**：算不清无形收益

---

## #14 期望值思维
- **来源**：西方（帕斯卡 → 概率论）
- **核心问题**：哪个选择期望收益最大
- **核心动作**：概率 × 收益 = 期望值

**思考流程**：
1. 列出每个选项的可能结果
2. 估算每个结果的概率
3. 估算每个结果的收益/损失
4. 计算期望值：Σ(概率 × 收益)
5. 选期望值最高的（或根据风险偏好调整）

**关键提问**：
- 最好的情况概率多大？最坏的呢？
- 如果重复这个决策100次，平均结果是什么？
- 我的概率估计是否被情绪影响了？

**组合接口**：#03 贝叶斯思维、#11 二阶思维、#13 机会成本
**反面风险**：概率估计不准导致误判

## #15 决策树
- **来源**：西方（决策分析/运筹学）
- **核心问题**：有哪些路可以走
- **核心动作**：拆出所有路径和分支，逐一评估

**思考流程**：
1. 定义决策点
2. 拆出所有可选路径
3. 每条路径继续拆出后续分支
4. 标注每个节点的概率和收益
5. 从后往前推算最优路径

**关键提问**：
- 在每个节点，有哪些选择？
- 每条路径通向什么结果？
- 哪些分支是不可逆的？

**组合接口**：#14 期望值思维、#16 先为不可胜、#20 多路径对比
**反面风险**：树太复杂，反而混乱

## #16 先为不可胜
- **来源**：华夏（《孙子兵法·形篇》）
- **核心问题**：最坏能输多少
- **核心动作**：先立于不败之地，再想怎么赢

**思考流程**：
1. 评估最坏情况
2. 确保即使最坏情况发生也能承受
3. 在安全底线之上，再追求收益
4. 设计"输了也不死"的方案

**关键提问**：
- 最坏会怎样？我能承受吗？
- 有没有退路？
- 在不输的基础上，怎么赢得更多？

**组合接口**：#11 二阶思维、#53 留有余地、#57 最坏情况推演
**反面风险**：过度保守，错失机会

---

## #17 中庸之道
- **来源**：华夏（《中庸》）
- **核心问题**：该做到什么程度
- **核心动作**：量化"度"，找到刚刚好的平衡点

**思考流程**：
1. 识别两端（过与不及）
2. 分析两端各自的利弊
3. 寻找最优"度"的位置
4. 设定可衡量的"度"的指标

**关键提问**：
- 做到什么程度就够了？过了反而不好？
- 过度和不足，哪个风险更大？
- 有没有办法量化这个"度"？

**组合接口**：#12 阴阳平衡、#19 反向思维、#53 留有余地
**反面风险**：变成平庸，缺乏锐度

---

## #18 延迟决策
- **来源**：西方（决策理论/信息经济学）
- **核心问题**：现在必须决定吗
- **核心动作**：用时间换信息，在信息不足时等一等

**思考流程**：
1. 评估当前信息充足度
2. 判断等待能获得什么额外信息
3. 评估延迟的成本（错过窗口？）
4. 如果等待收益 > 延迟成本 → 延迟

**关键提问**：
- 如果再等一个月，我会知道什么现在不知道的？
- 延迟的代价是什么？
- 有没有"不可延迟"的硬性截止？

**组合接口**：#03 贝叶斯思维、#42 天时判断、#44 后发制人
**反面风险**：等太久错失窗口

## #19 反向思维
- **来源**：西方（查理·芒格）
- **核心问题**：怎么才不会搞砸
- **核心动作**：先想"如何失败"，从反面提取洞察

**思考流程**：
1. 问"如何让这个事情彻底失败？"
2. 列举所有"愚蠢"的做法
3. 从反面提取"应该避免什么"
4. 正向应用：避免这些就提高了成功率

**关键提问**：
- 如果要故意失败，会怎么做？
- 别人在类似情况下是怎么搞砸的？
- 我现在做的，有没有"搞砸清单"上的元素？

**组合接口**：#12 阴阳平衡、#16 先为不可胜、#57 最坏情况推演
**反面风险**：陷入悲观，不敢行动

---

## #20 多路径对比
- **来源**：西方（战略规划/选项分析）
- **核心问题**：有没有第三种方案
- **核心动作**：同时看3种以上选择，横向对比

**思考流程**：
1. 列出当前看到的选项（通常是2个）
2. 强制发散：还有没有第3、4、5种？
3. 对每个选项评估：成本、收益、风险、时间
4. 横向对比，找最优解或组合方案

**关键提问**：
- 除了A和B，还有C吗？
- 能不能组合A和B的优点？
- 每个选项的隐含假设是什么？

**组合接口**：#13 机会成本、#15 决策树、#20 多路径对比
**反面风险**：选项太多，选择瘫痪
