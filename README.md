# 问道 (Deep Thinker) 🧘

**不给答案，带你把问题想清楚。**

问道是一个教练式深度思考技能——68种结构化思维元件 + 中西融合方法引擎 + 教练式交互流程。不是告诉你结论，而是带你走一遍思考过程。

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| **68个思维元件** | 7层结构：认知/决策/行动/关系/时间/风险/系统 |
| **中西融合** | 华夏24框架（格物致知/庖丁解牛/法术势/阴阳平衡…）+ 西方44方法（第一性原理/二阶思维/贝叶斯…） |
| **三种参与模式** | 💡轻量点拨（1句话+反问）→ 🧭标准分析（5模块）→ 🔬深度研究（假设审计+多路径推演） |
| **教练式交互** | 不给答案，用问题驱动思考。输出永远包含"结构+洞察+选择" |
| **反机械化** | 表达约束/叙事弧线多样化/同类方法去重/长对话收束 |
| **Agentic研究** | Step 0自动判断是否需要外部信息，需要就去搜，搜完再分析 |

## 🧠 七层思维元件

```
① 认知层（#01-#10）——怎么看
② 决策层（#11-#20）——怎么选
③ 行动层（#21-#30）——怎么做
④ 关系层（#31-#40）——怎么看人
⑤ 时间层（#41-#50）——怎么看时间
⑥ 风险层（#51-#60）——怎么防错
⑦ 系统层（#61-#68）——怎么看整体
```

每个元件包含：名称、来源（华夏/西方）、核心动作、组合接口、反面风险。

## 🚀 使用方式

### 标准模式（默认）
```
用户："我该不该辞职创业？"
问道：[问题重构] → [思考路径：结构→人心→时间] → [选择路径：ABC] → [点醒一句] → [轻反思]
```

### 轻量模式（日常对话）
```
用户："我觉得这个方向没问题"
问道："这个方向成立的前提是XX——你确认这个前提对吗？"
```

### 深度模式
```
用户："帮我彻底想清楚这件事"
问道：5模块 + 假设审计 + 多路径未来推演（1年/3年/5年）
```

## 🎯 适用场景

- 战略判断、重大决策
- 日常对话中的判断/选择/纠结
- 任何需要"把问题想清楚"的场景
- 创业/商业/产品/团队/人际关系/投资/个人成长
